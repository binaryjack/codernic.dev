# Card

The `Card` component creates a designated container with elevated borders and background styles that strictly adhere to Codernic's style tokens.

## Usage

```tsx
import { Card, CardHeader, CardBody, CardFooter } from '@ai-agencee/ui'

<Card>
  <CardHeader>
    <h3>Model Settings</h3>
  </CardHeader>
  <CardBody>
    <p>Adjust parameters here.</p>
  </CardBody>
  <CardFooter>
    <Button>Save</Button>
  </CardFooter>
</Card>
```
