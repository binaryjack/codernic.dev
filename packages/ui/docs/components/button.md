# Button

The `Button` component is the primary call to action in Codernic UI.

## Usage

```tsx
import { Button } from '@ai-agencee/ui'

<Button variant="primary" size="md">Click Me</Button>
<Button variant="ghost" size="sm">Cancel</Button>
<Button variant="danger" size="lg">Delete</Button>
```

## Variants
- `primary`: Solid brand color for primary actions.
- `secondary`: Subtle outline for alternative actions.
- `ghost`: Transparent background, useful for icon buttons.
- `danger`: Red destructive action.
